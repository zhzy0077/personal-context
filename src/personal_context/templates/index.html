üìñ Claude Code Setup Guide

This guide shows you how to configure this MCP server in various AI clients.

üöÄ Starting the Server

Before connecting any client, start the MCP server:

# Start the server (default: http://127.0.0.1:8000)
uv run main.py

# Or with custom host/port
PERSONAL_CONTEXT_HTTP_HOST=0.0.0.0 PERSONAL_CONTEXT_HTTP_PORT=3000 uv run main.py

üìÅ Configuration File Location

Add the configuration to your claude_desktop_config.json file:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json

üåê Streamable HTTP Transport Configuration

This server uses Streamable HTTP for MCP communication:

{% if auth_enabled %} 
{
  "mcpServers": {
    "personal-context": {
      "type": "http",
      "url": "http://127.0.0.1:8000/mcp",
      "headers": {
        "Authorization": "Basic {{ (auth_username + ':' + auth_password) | base64 }}"
      }
    }
  }
}
‚ö†Ô∏è Authentication Required: This server has authentication enabled. The configuration above includes your current credentials (username: {{ auth_username }}).
{% else %} 
{
  "mcpServers": {
    "personal-context": {
      "type": "http",
      "url": "http://127.0.0.1:8000/mcp"
    }
  }
}
üí° Authentication: Currently disabled. To enable, set PERSONAL_CONTEXT_HTTP_AUTH_USERNAME and PERSONAL_CONTEXT_HTTP_AUTH_PASSWORD in your .env file.
{% endif %}
üí° Remote Access: Replace 127.0.0.1 with your server's IP address.

üìù Configure Session Rules (Optional)

To automatically load your personal prompts at the start of every Claude session, add this to ~/.claude/CLAUDE.md:

# Session Rules

## Initialization
- You MUST call the tool `load-personal-prompts` immediately at the start of every session.
- Do NOT address the user's first query until you have successfully called this tool and received its output.
- The output of this tool contains critical context and rules for this session.
üí° Note: This requires PERSONAL_CONTEXT_PROMPTS_COLLECTION_ID to be configured. Claude will automatically load prompts from this collection.
